1. Introdução

O HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas da web. Ele define a estrutura e o conteúdo de uma página, como títulos, parágrafos, links e imagens.

Através do HTML, é possível organizar os elementos de uma página de forma hierárquica, utilizando tags específicas. Essas tags são interpretadas pelo navegador para exibir o conteúdo de maneira estruturada.

2. Tags Básicas

As tags HTML são elementos fundamentais que dizem ao navegador como exibir o conteúdo. Algumas das mais usadas incluem <h1> a <h6> para títulos, <p> para parágrafos e <a> para links.

Cada tag geralmente possui uma abertura e um fechamento, como <div></div>, delimitando o início e o fim de uma seção ou conteúdo específico na página.

3. Atributos

Os atributos das tags permitem adicionar informações adicionais a um elemento HTML. Por exemplo, o atributo href na tag <a> define o destino de um link, enquanto o atributo src em uma tag <img> especifica a localização de uma imagem.

Esses atributos são utilizados para controlar o comportamento e a aparência dos elementos HTML, tornando as páginas mais dinâmicas e personalizadas.

4. Formatação

Com o HTML, você pode controlar a formatação básica do texto em uma página web. Tags como <strong> e <em> são usadas para negrito e itálico, respectivamente.

Além disso, é possível criar listas ordenadas com <ol> e listas não ordenadas com <ul>, facilitando a organização de informações em formato de lista.

5. Formulários

Formulários em HTML permitem a interação do usuário com a página. Usando a tag <form>, é possível capturar informações inseridas em campos de texto, caixas de seleção, botões de rádio e muito mais.

Cada formulário pode ser enviado para um servidor para processamento, utilizando diferentes métodos, como GET ou POST, dependendo do tipo de informação que está sendo transmitida.
